# Block centreren

Hoe centreer ik een block-element, bijvoorbeeld een <div> midden in de pagina, of alle inhoud van een webpagina?

Een stukje HTML:

```
<body>
  <div id="bedenk-je-eigen-id">Dit block moet gecentreerd worden</div>
</body>
```

Een stukje CSS:

```
# bedenk-je-eigen-id {
  width: 200px;
  margin: auto;
}
```
